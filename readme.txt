1.js简介
    - js是一种编程语言，解释执行（脚本语言），可用于客户端和服务端
    - 用于网页开发，可以直接嵌入HTML代码中，由浏览器解释执行
    - 用于服务器端开发，如node.js，借助v8引擎执行
    - 编程初学者为什么选择js作为第一门编程语言
        - 语法简单，容易学习
        - 代码执行在浏览器中，不需要配置环境， 可以直接看到效果
        - 全栈开发
